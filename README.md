# Neighborhood Smart Contracts

## Development

### Install Dependencies

```shell
npm i
```

### Test the Contracts

```shell
npx hardhat test
```

### Deploy the Contracts

```shell
npx hardhat run --network {network} scripts/deploy.js
```

### Generate UML Diagram

Install UML tool

```shell
npm link sol2uml --only=production
```

Run UML tool

```shell
sol2uml ./contracts -r -f png
```
